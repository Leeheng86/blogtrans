Depend on:

GData python client library (for blogger)
http://code.google.com/p/gdata-python-client/



Data structure: Similar to MTIMPORT structure



AUTHOR :string
TITLE :string
DATE :date
PRIMARY CATEGORY :string
CATEGORY : [strings] (could be multiple)
STATUS :string (publish, draft)
ALLOW COMMENTS :0 or 1
ALLOW PINGS :0 or 1
CONVERT BREAKS :??

BODY :string (multi-line)
EXTENDED BODY :string (multi-line)
EXCERPT :string (multi-line)

[COMMENT]
  AUTHOR :string
  EMAIL :string
  URL :string
  IP :string
  DATE :string (multi-line)

[PING]
  TITLE :string
  URL :string
  IP :string
  BLOG NAME :string
  DATE :date
